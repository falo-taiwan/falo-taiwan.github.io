Class 03 Topic 09

主題九：Vibe Coding 實戰武器庫

盤點並分類 Class 03 核心實戰工具，為 Class 04 系統整合奠定工具地基

AI 程式開發與原型代理 (Coding & Prototyping)

Antigravity / Codex Agent
具備地端終端指令執行與精密檔案 Diff 補丁修改能力的 AI 代理。
實務重點：自動化執行規劃、執行與驗收三部曲，避免整檔覆寫造成的 Token 浪費與錯誤。
Claude Code Agent
Anthropic 推出的 Terminal 命令行代碼代理人，直接對本地代碼庫進行對話式編輯。
實務重點：在 CLI 下快速發起程式重構、直接執行本機單元測試並自動產生 Git 提交。
Claude Artifacts / v0.dev Prototyping
網頁端 UI 快速互動原型設計工具，可即時渲染出前端元件畫面。
實務重點：在進階寫檔前，先透過對話調整出漂亮的前端網頁，再將代碼直接匯出以節省刻介面時間。

知識工程與即時研究 (Knowledge & Research)

NotebookLM Refinery
專案理解與知識提煉煉油廠。
實務重點：倒入 legacy 程式碼、SQL DDL、ERP 手冊等「原油」，提煉出精準問答、FAQ 或中文大綱，做為 AI 輔助規格提煉。
Perplexity Research
即時聯網技術搜尋引擎與 AI 瀏覽器。
實務重點：秒查第三方 API 最新官方規格、第三方依賴改版細節或特定的程式報錯解決偏方。

地雲整合與通信橋樑 (Integration & Bridging)

Google Apps Script (GAS) Cloud Server
Google 生態系輕量 Serverless API 後端，支援無縫存取 Sheet、Mail 與 Calendar。
實務重點：藉由 HTML Fetch API 連接 GAS，實現網頁前端數據即時寫入雲端 Sheet 的超低阻礙對接。
地端 HTML Server & Python 呼叫 Local Server
地端 full-stack 自動化架構，以 Web UI 觸發本機 Python 執行作業系統層任務。
實務重點：使用 HTML 按鈕觸發 Python 呼叫本機 CLI、執行大宗檔案重新命名或 OCR 辨識排程。
ngrok / LocalTunnel Tunnel
地端 Webhook 穿透聯測工具。
實務重點：將地端 `localhost` 服務秒變 HTTPS 公網網址，以便在本地直接接收 LINE Bot 訊息或外部 ERP webhooks 回傳，省去雲端部署時間。
