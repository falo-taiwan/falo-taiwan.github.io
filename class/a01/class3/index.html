Tianxin ERP Internal Syllabus

Class 03 課程主題導航

探索 AI Agent 開發、Vibe Coding 以及如何將維護經驗提煉封裝為天心 ERP 組織能力資產

Topic 01

人與 AI 的共創共生

探討 AI 時代的「協作與治理觀念」基礎。理解為什麼我們不是被取代，而是學習與 AI 助理、顧問、副駕駛進行高效人機協作 (HITL) 與放大個人能力。
進入主題一
Topic 02

AI Agent 協同討論與即時知識工程化

解析「對話即生產」與動態共創筆記模式。理解如何打破傳統靜態知識管理限制，讓 AI Agent 在討論中即時進行結構化寫檔與決策軌跡記錄。
進入主題二
Topic 03

AI Agent & Vibe Coding 工具分析與對比

橫向對比四大開發模式（Antigravity、Claude Cowork、ChatGPT Codex、Gemini Canvas 模式），著重探討上下文視窗、本地操作能力、Diff 區塊替換等核心指標，助您挑選最適合的 Vibe Coding 工具。
進入主題三
Topic 04

AI 時代的知識資產化案例實踐

以「Chrome 內建 Gemini 解鎖」為實戰案例，展示如何遵循 FALO 六層模型，將一個真實問題提煉為組織的能力包 (Capability Pack)，並用 NotebookLM 扮演知識煉油廠。
進入主題四
Topic 05

OCR 辨識策略與 HITL 人機協作實務

以「郵務自動化分揀辨識延誤」為實戰案例，分析自動化系統的辨識邊界、校對 (Audit) 設計與 HITL (Human-in-the-Loop) 的必要性，並探討其在銀河 ERP 電子發票與排程系統中的應用。
進入主題五
Topic 06

從社群痛點到 AI 代理協作：LINE 防封鎖工具與 Computer Use 模擬實務

解析 LINE 防封鎖工具的開發歷程。展示如何從社群真實問題出發，經由 Gemini Canvas 快速 PoC、Antigravity 高規進化，再到文件變形輸出與 Computer Use 模擬操作之完整知識工程實踐。
進入主題六
Topic 07

學員課前提問與技術解答 (QA)

收錄並解答學員提出的 12 個核心技術提問，包含高併發 API 設計、資料庫命名管理、Token 用量分配、Docker 環境隔離、安全掃描以及 ERP App 開發決策。
進入主題七 (QA)
Topic 08

Chrome 外掛程式載入與部署實戰教學

使用 FALO Instagram Video Enhancer v1.0.1 作為範例，手把手學習如何在開發者模式下部署、載入並測試 Chrome 擴充功能，包含智慧遙控器、5 種色彩主題與鍵盤快捷鍵的實作。
進入主題八
Topic 09

Vibe Coding 實戰武器庫：地雲整合與 AI 開發工具盤點

盤點並分類 Class 03 核心實戰工具，涵蓋 AI 原型開發（Antigravity、Claude Code）、知識提煉（NotebookLM、Perplexity）與地雲整合實作（GAS、地端 Python 伺服器、ngrok 穿透），為 Class 04 系統整合奠定地基。
進入主題九
Topic 10

從 ERP 外掛到 AI Agent

理清企業自動化演進六階段（人工 ➔ ERP 客製 ➔ RPA ➔ Workflow ➔ AI Agent ➔ Multi-Agent）。立足天心 ERP 顧問熟悉之 Workflow 模式（Event-Rule-Action），探討白箱架構 n8n 在 AI 治理中的全新優勢與 AI Review Workflow 新觀念。
進入主題十
